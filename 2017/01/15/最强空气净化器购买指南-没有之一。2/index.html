最强空气净化器购买指南，没有之一(2/2)

发表于 2017-01-15   |   分类于 环境&生存   |     |   次阅读

有了以上(1/2)的理论基础，我们终于可以开始选择一台适合自己的空气净化器了。

按照选购要点一步一步进行，很容易就能得出结论了。

1、净化器类型

这里curt帮你做主了，只选择过滤型。

2、使用需求

最重要的是使用面积（也就是室内的平米数，根据面积*10（15）来确定需要的CADR值）。

另外除了去除颗粒物（PM10/PM2.5），是否有去除烟味等异味，或是装修除甲醛等挥发性有机物的需求。

以及家中是否养宠物，是否需要除菌功能等。不仅宠物，人身上的脱落物和病菌也是污染源哦.

3、量化数据“双C”

找到一款空气净化器的CADR和CCM值，同时也要确定数据的可信度，例如厂家给出的检测机构认证书。

4、使用成本

由于空气净化器需要长时间运行，功耗（用电量功率/W，净化能效m3/W/h）也是需要考虑的问题，同时耗材（各种滤网）的更换周期和更换成本也很重要，还要考虑使用过程中的维护费用（厂家的售后服务）。

5、噪声

噪声是过滤式净化器不可避免的问题，毕竟时刻都有大风量进出，即使厂家会使用各种吸音，隔音的方法，开启空气净化器仍然会给家里加一个背景音。

一台净化器长时间持续稳定工作时的噪声和空气严重污染最大功率运行时的噪声都是要考虑的因素，一般对于不同型号的净化器厂家也会表明运行时的分贝数（dB）作为参考。

6、附加功能/人性化

包括手机APP操控等附加功能可以提高易用性，但智能运行等自动功能的准确性需要特别注意.

很多厂家会为净化器添加利于使用的附加功能，这里举几个常见的例子：

1)【智能检测】

很多空气净化器都可以实时检测空气质量，但仅限于机器附近，不能涵盖整个房间所以只具有参考价值。

自动模式/智能运行，这点尤为重要，毕竟人还是很懒的，也会遇到外出自己无法控制，或是家中老人不太在意的情况。

针对智能检测，很多型号的空气净化器提供了自动模式，根据检测到的空气质量自己调整运行模式，这就要求检测结果要相对准确。还要注意的是，自动模式设定的空气质量目标是否可以调整，或是是否在一个较高的空气质量范围。否则始终处于空气质量一般的智能运行也没有意义。

2)【静音/睡眠模式】

有些人对噪声比较敏感，睡觉的时候需要较低噪声运行。一般厂家为了宣传，往往会使用最大档位的CADR和较小档位的噪声，一定要注意在睡眠模式或低噪声运行时净化效果是否也有保证。

无光运行/低亮度，这其实是个很小的细节，但是也很重要，想想那些跑马灯充电器，呼吸灯鼠标，在黑暗环境下的光污染让人头疼。

3)【滤网更换】

是否有清晰易懂的滤网状态提示，可以及时更换避免二次污染或净化效率下降。另外通常情况下更换滤网是需要自己操作的，更换的难易程度也很重要，这样考虑，模块化设计更有优势。

4)【售后服务】

这点是所有电器都要考虑的问题，主要跟厂家有关，curt建议尽量选择主流品牌的主流型号，避免由于过于小众而被厂家抛弃成为三不管。

此外，入风出风方式，摆放位置和占地，移动是否方便，颜值是不是可以接受等，都是需要考虑的范围。而一般来说所谓的“新技术”“新名词”不要盲目的追求，更务实的CADR和CCM值才是硬道理。

7、价格

空气净化器不便宜，毕竟质量和功能服务都是要用钱来买的，设定自己的预算，重视自己需求更高的功能，在不差钱的情况下，可以选择CADR值和CCM值更高的产品。

8、仅供参考的产品推荐

专注做空气净化器的品牌（如欧美的bluair，飞利浦，日本的夏普，松下等）都有会比较完整的产品线，覆盖不同室内面积的使用需求（不同的CADR值），提供比较多样的选择。CADR值越高，意味着价格也越贵，一般情况，适用于大面积的高CADR型号用在小房间也是可以的，反之则不建议。

很多品牌也会大肆宣传自己的专利技术或特殊滤网，有时候会用上一些让人听起来就很厉害的术语比如“等离子”，“光触媒”等，可能只是宣传的噱头，无需刻意追求，还是应该坚持本心按照我们给出的选购流程选择。

国产和进口品牌也会有自己的不同特点：

欧美品牌往往是特殊用途（医用，工厂用）转为做家用净化器，各自特点也比较鲜明。

日韩品牌产品线非常丰富，而且经常有很多加湿等附加功能。

国内市场现在是一片鱼龙混杂的状况，各家的宣传和噱头看起来也都很厉害，选择的时候还是要注意我们提到的各个要点，如果找不到相关的信息，净化效果就要打个问号（一分钱一分货还是有道理的）。

国产品牌里有一个小米这样的搅局者，curt认为不失为一件好事。

结语

就curt自身选购体验来说，双C量化指标可以作为重要参考，但其可信度值得商榷（尤其是国产品牌）。

空气净化器最关键的部件是滤网，再强大的附加功能都不如实实在在的高效滤网，而在这点上，用户无法测试判断实际滤网的等级，无奈之余，curt建议选择知名品牌会靠谱点，相信品牌的力量吧。

滥竽充数的滤网，不仅起不到净化空气的功效，而且由于长期滋生细菌灰尘，造成环境空气的二次污染，还不如不用。（一分价钱一分货，厂家都不傻）

既然选择使用了空气净化器，就别舍不得滤网耗材的更换费用。
