以一个外行者，看六大茶类的区别

发表于 2017-01-14   |   分类于 茶道   |     |   次阅读

当然，在众多的茶类划分中，运用最广泛、最权威、认知度最高的当属中国六大茶类。

即绿茶、红茶、青茶、黑茶、白茶、黄茶。

绿茶

绿茶是一种不发酵的茶。

  • 【特点】绿茶具有香高、味醇、形美、耐冲泡等特点。
  • 【制作工艺】其制作工艺都经过杀青、揉捻、干燥三步骤的过程。

杀青方式有加热杀青和热蒸杀青两种，以蒸青汽杀青制成的绿茶称“蒸青绿茶”。

干燥，以最终干燥方式不同有炒干、烘干和晒干之别，最终炒干的绿茶称“炒青”，最终烘干的绿茶称“烘青”，最终晒干的绿茶称“晒青”。

  • 炒青绿茶：龙井（杭州西湖）、大方（安徽歙县）、碧螺春（洞庭茶）、高桥银峰（湖南长沙）、雨花茶、松针（吉林）等
  • 烘青绿茶：黄山毛峰(徽）、太平猴魁（徽）、华顶云雾（浙江天台山）等
  • 晒青绿茶：滇青、川青、陜青等
  • 蒸青绿茶：煎茶、玉露等

红茶

红茶是一种全发酵的茶，发酵度为80～90%。

  • 【特点】红汤红叶。
  • 【制作工艺】基本制作工艺流程是萎凋、揉捻、发酵、干燥。

红茶与绿茶的区别，在于加工方法不同。红茶加工时不经杀青，而用萎凋使鲜叶失去一部分水分，再揉捻(揉搓成条或切成颗粒)，然后发酵，其实质是茶叶中原先无色的多酚类物质，在多酚类氧化酶的催化作用下，氧化以后形成了红色的氧化聚合物——红茶色素。这种化合物一部分溶于水，一部分不溶于水，而积累在叶片中，从而形成红汤、红叶。

红茶主要有小种红茶、工夫红茶和红碎茶三大类。

  • 小种红茶：正山小种（武夷山）、烟小种等
  • 工夫红茶：滇红（云南）、金骏眉（武夷山）、祁红（安徽）、川红、闽红等
  • 红碎茶：叶茶、碎茶、片茶、末茶等

青茶（乌龙茶）

青茶是属半发酵的茶，其发酵度为30～60%。

在制作时使其适当发酵，让叶片稍有红变，是介于绿茶与红茶之间的一种茶类。它既有绿茶的鲜浓，又有红茶的甜醇。其叶片中间为绿色，叶缘呈红色，故有「绿叶红镶边」之称。

  • 闽北乌龙：武夷岩茶、水仙、大红袍、肉桂等
  • 闽南乌龙：铁观音（福建安溪）、奇兰、水仙、黄金桂等
  • 广东乌龙：凤凰单枞、凤凰水仙、岭头单枞等
  • 台湾乌龙：冻顶乌龙、包种、乌龙等

白茶

白茶属轻度发酵的茶，发酵度为20～30%，是我国的特产。

  • 【制作工艺】基本工艺过程是萎凋、晒干或烘干。

在加工时不炒不揉，只将细嫩、叶背满是茸毛的茶叶晒干或用文火烘干，而使白色茸毛完整的保留下来。

如福鼎大白茶，芽壮多毫，制成的成品茶满披白毫，十分素雅，汤色清淡，味鲜醇。

白茶主要产于福建的福鼎、政和、松溪和建阳等县。

  • 白芽茶：白毫银针（福建）等
  • 白叶茶：白牡丹（福建）、页眉等

黄茶

黄茶是微发酵的茶，发酵度为10～20%。

  • 【特点】黄汤黄叶
  • 【制作工艺】加工工艺为杀青、揉捻、闷黄、干燥。

黄茶品质特点是“黄汤黄叶”，这是制茶过程中进行闷堆渥黄的结果。

黄茶主要的代表名茶有蒙顶黄芽、黄山黄芽、君山银针等。

  • 黄芽茶：君山银芽（洞庭君山岛）、四川雅安、蒙顶黄芽（四川蒙顶山）、霍内芽等
  • 黄小茶：北毛尖（信阳毛尖）、湖南的沩山毛尖、浙江的平阳黄汤、温州黄汤等
  • 黄大茶：广东大叶青、霍山黄大茶等

黑茶

黑茶属后发酵的茶，其发酵度为100%。

  • 【制作工艺】基本工艺流程是杀青、揉捻、渥堆、干燥。

一般原料粗老，加工时堆积发酵时间较长，因而叶色油黑或黑褐，故称黑茶。

黑茶主要供边区少数民族饮用，所以又称边销茶。是藏、蒙、维吾尔等兄弟民族不可缺少的日常必需品。有”湖南黑茶”、”湖北老青茶”、”广西六堡茶”、四川的”西路边茶””南路边茶”、云南的”紧茶”、”扁茶”、”方茶”和”圆茶”等品种。

  • 湖南黑茶：安化黑茶等
  • 湖北老青茶：蒲圻老青茶等
  • 四川边茶：南路边茶、西路边茶等
  • 滇桂黑茶：普洱茶（云南）、六堡茶等
