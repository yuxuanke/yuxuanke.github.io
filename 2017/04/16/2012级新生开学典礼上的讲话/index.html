中国计量学院林建忠校长 | 2012级新生开学典礼上的讲话

发表于 2017-04-16   |   分类于 中国计量大学   |     |   次阅读

同学们，老师们，家长们，Ladies and gentlemen：

Good evening！

在我们的地球上有这么一所学校，它的校名既张扬又低调。

说其张扬，是因为校名前两个字竟然是地球上人口最多国家的国名；说它低调，是因为纵然它涵括了八大学科门类的四十八个专业以及拥有两万多学生，校名的后两字却依然为学院。如果这一前一后还不能给出唯一答案的话，那么校名中间的两个字绝对是地球上唯一的。

它，就是中国计量学院！

今天，这所张扬而又低调的学校，在这秋高气爽的傍晚，迎来了意气风发的2012级新生。

首先，请允许我代表学校对各位同学如愿以偿地成为大学生和研究生表示衷心的祝贺！向从世界各地汇集而来的你们表示热烈的欢迎！向长期以来精心培育你们，并把你们托付给中国计量学院的家长们表示诚挚的谢意！

同学们，当你们在填报高考志愿和报考学校的时候，可能会中国计量学院的“计量”两字感到迷茫。

计量是什么？它重要吗？和测量有什么关系？测量和计量既有关联又有区别，这里仅举一个例子说明其重要性。

前不久，中国和菲律宾两国就黄岩岛的归属问题产生了对峙，而证实黄岩岛是中国领土的有力证据之一就是13世纪我国科学家郭守敬曾经进行的四海测量，当时他在南海的测量点就是黄岩岛。说明中国在那时就已发现了黄岩岛，而国际法对于领土取得的四大要素之一就是最早发现。

计量竟然可以对祖国领土的完整起到如此关键的作用，可见它是多么的重要，所以你们选择中国计量学院是正确的！

同学们，当你们填报完志愿后，可能就会关注中国计量学院，也许就会知道它建校的时间并不长，只有34年。跟一些百年老校相比，它显得很年轻。

诚然，一般而言，建校时间越长，基础也就可能越雄厚，实力也就可能越强。但是，大学是人类社会文明的产物和标志，不同时期的文明，产生不同的大学。

在100年以前，甚至50年以前，是不可能产生以计量为主的大学的，因为人类社会文明还没有发展到这种程度。所以，中国计量学院是人类文明进入到一个较高阶段的产物，且是世界上唯一的产物。

正所谓，“校不在老，稀罕则名”。所以，你们选择中国计量学院是明智的！

同学们，当你们拿到录取通知书之后，就意味着在今年秋天，你们要告别熟悉的故土，离别身边的亲人，只身来到一个陌生的地方。

“自古逢秋悲寂寥”，更何况要离别，所以，感伤和惆怅也许会在一段时间萦绕着你们。但是，同学们，我相信这种离别愁绪不会持续太久，你们很快就会从中解脱出来。因为虽然你们离开了故土，但是来到的是中国计量学院这一方热土；你们告别了亲人，见到的是计量学院这个大家庭新的家人。

我相信，在不久的将来，你们就会感受到这个大家庭的和谐和欢乐！

同学们，当你们跨进中国计量学院大门的时候，新的生活就开始了。

希望在将来的几年中，通过老师的言传身教，通过计量学院所特有的以“天地观”为基点的环境熏陶，你们能够成为有大气象、大格局而又脚踏实地的人。

这种大气象、大格局包含着对真理、正义、自由、博爱的追求之志，对国家、民族、社会、民众的关切之情。气象一大、格局一大，内心就宏阔，精神就逍遥，灵魂就会奔逸自由。

那么，以“天地观”为基点的环境是什么呢？就让我来带领大家领略一下“八天”图吧！

  • 天外是宇宙——校园西区的环宇楼、翔宇楼，东区的广宇楼是教学楼。

希望同学们今天在课堂上环绕、翱翔和拓广知识的宇宙，增加自己的智慧；明天走上社会之后，能够环绕宇宙，飞得更远；翱翔宇宙，飞得更高；拓广宇宙，飞得更快！

  • 天机犹可问——学校的天问科技活动中心是学生课外实践活动的场所。

在2300多年前，伟大的浪漫主义诗人屈原创作了《天问》，在《天问》中，他就天、地、自然、社会、人生、历史提出了173个问题。希望今天你们在天问中心，以一种天问的姿态消化知识、消除疑惑、小试身手；明天则在社会的广阔天地里大显身手！

  • 天行存刚健——学校的天健体育馆是健身的场所。

《易经》中有“天行健，君子当自强不息”。自然的运动刚劲强健，有生命的人类就更应当如此。如果我们今天能够以君子为范，强身健体，磨练意志，积蓄更多的正能量，那么，明天就一定能如虎添翼，刚毅坚卓，无往而不胜！

  • 天空行日月——学校的日月湖是很可贵的一方水域。

湖面水波涟漪，湖畔树影婆娑，徜徉在这天人合一的境界中，大家可以闭上眼睛，享受物我皆忘的空灵。日月经天、江河行地，当然，你们也可以于此思考人生。

在今年的欧洲杯足球赛上，当意大利球星巴洛特利突破西班牙队球门的最后一道防线、卡西利亚斯坐等待毙之际，他花了两秒钟思考了人生，结果错失了射门的良机。而你们可以很从容地在湖畔思考，由此而孕育出惊天的豪气。

日月需护驾，在日月湖的边上，有两座亭子，叫羲和亭和望舒亭。“羲和”和“望舒”分别是为太阳和月亮驾车的神。它们安静地伫立在湖畔，构成了一幅曼妙的湖亭景色，启发人们用诗意的眼光去欣赏生命，用诗心的态度去对待生活。

同时，“羲和剑”与“望舒剑”是《仙剑奇侠传四》中琼华派用来网缚妖界吸取灵力的工具。希望你们在这里能够精心铸就出一把身心的守护之剑！

  • 天上有星辰——学校西区有长庚道、启明道、北斗路、星光路；东区有天权道、开阳道。

东启明、西长庚，都是金星的别名，分别出现在黎明与黄昏的天空。黎明与黄昏的交替，昭示着我们应当掌握生活的节奏，把握生命的韵律。同时，金星也叫Venus，它是爱与美的化身，需要我们毕生去追求。北斗，是古人用来辨别方向和确定季节的星座，它启迪大家要把握人生的方向，参透世态的炎凉。

天权是文曲星，代表着聪明才智；开阳，是武曲星，象征着刚毅果决、自强自立、不畏挫折。希望你们今天走在星光路上，明天能在各自的领域中成为耀眼的明星！

  • 天地呈方圆——学校有方圆楼。

“方圆之道”是中华文化所倡导的一种有理性的中庸之道。“方”，是立身的根基，方正的品质需要我们去培养、去守护；“圆”，是处世的智慧，圆融的技巧需要我们去揣摩、去践行。

希望大家能在天圆地方的大意境下去建立包容天地方圆的大情怀！

  • 天高需仰观——学校有仰仪楼。

“仰仪之态”，是要求我们去仰慕先哲之智慧，敬仰真理之圣洁。

校园里更有贤哲引路，这些贤哲不仅是伟大的科学家，而且其名字也很有寓意。

冲之路——曾经的圆周率精确度世界纪录保持者、数学家祖冲之激励我们朝着自己设定的目标奋力“冲之”。守敬道——元代科学家郭守敬勉励我们“守”道“敬”业。沈括路——中国古代百科全书《梦溪笔谈》的作者沈括劝励我们尽“括”知识于囊中。张衡道——东汉发明家张衡策励我们去创造一个均“衡”、幸福的人生！

  • 天道乃酬勤——学校西区有格致楼、求是楼，东区有行知楼、博雅楼。

勤者需格物致知，弄清事物的真相，懂得做人的道理；勤者需求是求真，要从科学、真理与理想三方面去追求客观事物的本来面貌、合理关系和发展前途；勤者需知行合一，将认识事物的道理与在现实中运用道理紧密地结合起来。勤者尚博学幽雅，要汲取广博的知识，涵养幽雅的性情。

只要我们今天勤奋努力，最大限度地完善和充实自己，就会有一个美好灿烂的明天！

同学们，在这秋高气爽的夜晚，在这量院生活即将开始的时刻，站在钱塘江畔，我仿佛看见了“潮头怒吼贯江面”的钱江大潮，仿佛感受到了那种奔放、汹涌、壮阔与奋进。

新的征程开始了，让我们整装待发，向着美好的未来进发！

谢谢大家！

2012年9月
