中国计量学院林建忠校长 | 2014届毕业典礼校长讲话：今天这个日子

发表于 2017-04-16   |   分类于 中国计量大学   |     |   次阅读

毕业生同学们、老师们、家长们：

上午好！

我致辞的主题是：今天这个日子。

今天是个特殊的日子。在人的一生中毕业的日子不多，而正是在2014年6月17日的这个日子，亲爱的毕业生们，你们大学毕业了！

今天是个喜庆的日子。大学的学习生活并不轻松，你们经过努力，圆满地完成了学业。在此，我代表学校向你们表示热烈的祝贺！

今天是个感谢的日子。你们在中国计量学院的几年里，学校获得了国家科技奖，年科研经费超过1个亿，你们不仅见证了学校的变化，更是学校发展的亲历者。

你们在全国比赛中，为学校获得了“挑战杯”一等奖、创业计划竞赛金奖、电子设计竞赛一等奖、机械设计大赛一等奖、电子商务挑战赛一等奖、航模锦标赛一等奖、数学建模比赛第5名、艺术展演合唱比赛一等奖等奖项；还获得了浙江省大学生足球赛冠军、舞蹈比赛一等奖、田径锦标赛总分第3等好成绩。

你们中有一位叫伊燕瑛的同学，4年的平均学分绩点是4.18；你们中有一位叫江航宇的同学，考研分数427，并以初复试总分第一的成绩被同济大学录取；你们当中还有一位叫魏乐的同学，他曾经入选吴克群团队参加了央视春晚的表演……还有很多，我就不一一例举了。

同学们，我为你们对学校发展所做出的贡献表示衷心的感谢！

今天是个感恩的日子。在过去的一千多个日日夜夜，学校的教职员工为同学们的学习和生活付出了辛勤的劳动，他们当中有循循善诱、一丝不苟的讲台能人；有孜孜不倦、春风化雨的指导高人；有知疼着热、无微不至的学工达人、有和蔼可亲、夜半开门的宿管亲人。

在此，我对为同学们的成长倾注了心血和希望的教职员工们表示诚挚的谢意！

今天是个伤怀的日子。当年你们来到了计量学院，一转眼就要各奔东西了，时间都去哪儿啦？春晚上王铮亮的“时间都去哪儿啦”唱出了对时间飞逝的感叹和浓浓的亲情。

最近中国计量学院版的“时间都去哪儿啦”这么唱到：“黑板前的老师啊，记忆中的面容啊，曾经同窗的那个他，前程路上再见吧”、“朝夕相处的日子，转眼就只剩下满满的离愁了”。

我们这些人今生在中国计量学院有过“交集”，这是一种缘分。如今，你们就要离校了，就要离开朝夕相处的老师、离开亲密无间的同伴、离开钱塘江边学源街上这片留下过你们笑声与泪水的土地。

我们今生有缘在计量，只要我们彼此永不忘，同学们让我们一起牢牢铭记吧，我们的心将永远相知与默契！

今天是个释怀的日子。学院版的“时间都去哪儿啦”中还唱到“校园老树不说话，满园蔷薇又开花。4年存了好多话，说不出些许牵挂”。

几年前，你们离开家乡，来到了这里，来到了第二个家。在家里，日月湖水映照过你们的身影，行知广宇留下过你们的足迹，嘉量启明回荡过你们的声音。在家里，学校的老师是爱你们的，但即便是爱，也难免有不周之处，你们或许也曾有过埋怨和不满。

然而，让我感到欣慰的是，我从你们那听到了临别时的呼声“你有你的紫金港、我有我的日月湖”！从这呼声中，我感受到了同学们对学校的一片深情。

同学们，让我们怀揣着这样的深情，祝愿明天的量院更加美好！

今天是个话别的日子。在你们临别之际，我有话要说，这些话不是简单的励志，在你们过去20多年的人生中，已经有过太多的励志教育。

你们是本科和硕士毕业生，是社会的知识精英，精英就要有精英的意识与作为，就要建立起精英应有的气场，因为当今生态环境的气场常被雾霾所笼罩，世态环境的气场常有暴戾之气、媚俗之风。

那么，要建立怎样的气场呢？

第一是骨气。人生在世，要活得有骨气。

骨气是什么？是一个人人格的体现，是一个国家国格的标志。

有骨气，才能具有高贵品性、才能不卑不亢、才能不畏强势、才能活出尊严、赢得尊重！

公元405年朝代更迭、社会动荡，在那年的秋天，有一个人来到了离家乡不远的彭泽县当县令。当年冬天有一位上面的官员来视察，这是一个既粗俗又傲慢的官员，一到彭泽县，就派人叫县令来拜见。县令得到消息，心里虽然很瞧不起这种发号施令的人，但也只得马上动身。正在这时，身边的人说，参见这位官员要十分注意小节，衣服要特别讲究，所谓的“束带见之”，态度要谦恭，不然的话，将来他会给你穿小鞋，你的官将保不住。一向正直清高的县令再也忍不住了，长叹一声说，我宁肯饿死，也不能因为这官衔向这样的人折腰。

于是，马上写了一封辞职信，离开了只当了83天的县令，从此再也没有做官。于是中国历史上少了一位地方官员，却多了一位文学家、诗人，这个人就是“采菊东篱下，悠然见南山”的陶渊明。

骨气是无条件要有的，不管你在世界上过得怎么样。

然而，谁不希望过得好一些呢？下雪的冬天，有炸鸡和啤酒，总比“心忧炭贱愿天寒”好，如果还能有一个都教授在身边那就更惬意了。

孟子说过，穷则独善其身，达则兼济天下。

如今，怎样才能既不穷，又要独善其身和兼济天下？这就引出了第二种气。

第二是底气。底气就是在这个世界上足以生存并且生存得好的能力。

四年一次的足球世界杯开始了，世界杯上众星云集。有这么一位球星，5岁开始在当地的街区少年足球队踢球，7岁时就初露锋芒，11岁时被诊断出患有阻碍骨骼生长的发育荷尔蒙缺乏症，但教练看中他的天赋，球队愿意出钱为他治疗，于是他13岁时来到西班牙的巴塞罗那，16岁就代表巴萨一队出场。他具有出众的球感、速度、爆发力、平衡能力、射门技术和阅读比赛的能力。在他的职业生涯中曾连续四年获得金球奖，获得过3次欧冠冠军、5次西甲冠军，还获得过世俱杯、世青赛以及奥运会冠军。

今年他与巴塞罗那完成了第7次续约，续约后他的年薪将达到2000万欧元，这位球星就是阿根廷人梅西(尽管我也喜欢C罗)。他在本次世界杯上已经打进了1个球，并且希望在这次比赛中圆他的世界杯梦。

这说明什么？说明技高一筹、能力出众，能在这个世界上左右逢源、游刃有余就是底气。

然而，要将外在的能力升华为内在的气质并由此提高外在的能力，还需要第三种气。

第三是锐气。锐气是一种胆量、是一种魄力。

有这么一个国家，它的国土仅比北京略大，但每一万人中就有140名科学家和技术人员，与这个国家相关的民族出现过马克思、爱因斯坦、弗洛伊德等著名人物。在1901年到2001年这100年间的诺贝尔奖获得者中，这个民族的人占了五分之一，而这个民族所占全世界人口的比例只不过五百分之一。

这个民族就是犹太民族，这个国家就是以色列。

这个国家有个当今最杰出的作家叫奥兹，他除了写小说，也时常发表对时势的见解。又一次，他收到总理府的来电，说总理读了他的文章，想邀请他一起喝咖啡，交流看法。他应邀去了，两人激烈辩论了一个半小时，结果谁也没有说服谁。

同样是这位作家，有一次乘出租车，上车后司机认出了他并说，我读过你的书，但是我不同意你的观点，接着便滔滔不绝地陈述了自己的观点，作家只有听的份。

这说明什么？说明广博知识、勤于思考、勇于创新、独立见解、坚持真理就是锐气，有了这样有锐气的民众，一个民族能有以上的成就也就不足为怪了。

一个人有了足够的底气和锐气后，就像容器中存在高压的气体，为了不膨胀爆炸，就要增大空间，于是就需要第四种气。

第四是大气。大气是一个人内心世界的外观表现，是对社会、生活、他人的理性接纳。

每个人的成长经历、知识结构、思维方式、品性脾气不尽相同，你得在这样的不同中搭建一个稳定的处世框架。这是一个有形和无形资源都短缺的社会，对待资源要有一个明智、淡定的姿态。

当年郑板桥从江苏兴化老家来到苏州，在城东桃花巷的东头开了画室，而在西头也有一个画室，主人是擅长画梅花的吕子敬。郑板桥到了苏州后只画兰竹、禽鸟、虫鱼，就是不画梅花，有人要他画梅花，他都介绍到西头的吕子敬画室，如此一来吕子敬更加得意，扬言苏州城里他说第二，没人敢说第一，这话传到郑板桥耳里，郑板桥一笑了之。

三年后，郑板桥离开苏州，临走时大家前来相送，吕子敬也来了。郑板桥当场酣畅淋漓地画了一幅梅花赠与吕子敬。吕子敬一看，惊得目瞪口呆地说道，郑兄之所以不画梅花，原来为的是给小弟我一口饭吃啊，惭愧，惭愧。

四种气讲完了。同学们，我相信，有了以上四种气场，你们就一定能气定神闲，笑傲江湖！

今天是个离别的日子。多情自古伤离别，毕业的季节总是泪眼朦胧。

同学们，我们就要分别了，从今以后，你们要凭一己之力去证明自己。在临别之际，我希望你们多保重。

不管你们将来身在何处，中国计量学院将永远和你们在一起。

你若安好，便是晴天！

谢谢！

2014年6月
