中国计量大学林建忠校长 | 离职演讲

发表于 2017-04-16   |   分类于 中国计量大学   |     |   次阅读

2017年4月12日，林建忠（校长）请辞中国计量大学校长。

当天，校长在闻厅进行了离职演讲，内容如下：

老师们，同学们：

我的离职宣布会于4月12日的8点在闻厅二楼报告厅举行，会上我发表了离别感言，于是就不向大家一一道别了。感言摘要如下，权当向大家道别。

2006年1月18日是一个冬雨绵绵的日子，这一天，我第一次来到位于钱塘江畔的下沙学源街258号，就任中国计量学院院长。
2017年4月12日是一个春和景明的日子，这一天，我离任中国计量大学校长，将回到浙大履职求是特聘教授。

十一年前来到这里，恰逢学校迎接本科教学评估，校的氛围与人的心境跟时令一样是隆冬季节。今天离开这里，校门口的中国计量大学校名正沐浴在和熙的春风中。

十一年零三个月，四千多个日子，我生命之河从临近”天命”流到了”耳顺”将至，中国计量大学也从临近”三十而立”到了将近”四十不惑”。

在过去的这些日子里，我们一起分享了学校教学评估优秀、荣获国家科技奖、国家教学奖、学校更名大学、学校排名迅速提升等众多喜悦，共同见证了中国计量大学的发展。今天的中国计量大学在全体师生员工的努力下已然”而立”，且不应”有惑”。

在这即将离别的时候，作为曾经的一校之长，我要对师生员工道一声感谢，谢谢你们为学校的发展所做出的努力，谢谢你们这些年来对我的信任、支持、理解与厚爱！

这些年来在校长的位置上，我为之奋斗的目标就是六个字：校发展，人幸福。

学校的发展具有阶段性且有无限的上升空间，人要幸福就要营造一个具有幸福感的环境，使每个人感受最小的压力，获得最大限度的快乐和自我实现。但感受最小压力和学校发展需要每人做出努力存在矛盾，学校资源往往不能满足自我实现的需求，这就会给人带来烦恼和郁闷，对此我深表遗憾，在此也深表歉意。

马上就要离开学校了，在闻厅这个会场，在嘉量会堂，我曾经数十次对毕业生挥手道别。今天，轮到我要说再见了。

对计量这片土地我有太多的感慨和留恋，对计量的师生我有太多的眷恋和不舍。无论将来怎样，计量都会是我魂牵梦绕的地方。

”人生自古伤别离”，为了缓解这样的伤感，我改动了仓央嘉措的诗聊以自慰：我在，或者不在计量，情就在那里，不来不去；我见，或者不常见大家，爱就在那里，不增不减。你在我的怀里，或者让我住进你的心里，默然，相爱，寂静，欢喜。

我虽然就要离开计量了，但我和小我二十岁的中国计量大学已经成了忘年之交。

今后的日子我会一如既往地关注学校的发展，祝愿中国计量大学蒸蒸日上！祝愿中国计量大学的师生们顺心如意，幸福安康！

谢谢

2017年4月
