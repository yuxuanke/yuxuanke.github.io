中国计量学院林建忠校长 | 2005级毕业生毕业典礼校长寄语

发表于 2017-04-16   |   分类于 中国计量大学   |     |   次阅读

亲爱的毕业班同学，老师们：

晚上好！

当六月的暑气吹浓了校园满地绿荫的时候，这就又到了毕业的时节。

人的一生当中或许有很多次毕业典礼，但是，大学毕业典礼无疑是最重要的一次。

因为，对于大多数毕业生而言，大学毕业就意味着结束了学生时代，走上了社会，这是人生的一个转折点。

因此，在这个重要的毕业典礼上，我想说说同学们的昨天、今天和明天。

昨天，同学们带着对大学生活的向往和对未来的憧憬，来到了西子湖畔，来到了钱塘江边，来到了中国计量学院。

几年来，同学们见证了学校的发展，经历了学校接受教育部本科教学工作水平评估并获得优秀的时刻，目睹了30周年校庆的盛况，你们是学校发展不可或缺的一员。

在此，我谨代表学校，对你们为学校作出的贡献表示衷心的感谢！

过去的四年，正是你们的情感趋于丰富、心智趋于成熟的加速期。多少个黄昏，当你们徜徉在日月湖畔时，是否有过和徐志摩同样的感受？“那河畔的金柳，是夕阳中的新娘，波光里的艳影，在我的心头荡漾”。

多少个日夜，当你端坐在翔宇楼和环宇楼时，是否曾想到终究有离别的那天？正是缘分，让我们与这片校园息息相关。这些天，在我们的校园里时常可以看到一群群身着学位服的同学在合影留念，一本本毕业纪念册在老师和同学的手中传递签名，这让我深深感受到同学们对同窗的友情、对老师的真情，更感受到了同学们对学校的深情。

在这难忘的一千多个日日夜夜，不管你曾经有过怎样的喜与怒、哀与乐，我相信，中国计量学院都将给你们留下永远美好的记忆！

今天，你们完成了学业，在这里举行毕业典礼，在这喜庆而又神圣的时刻，请允许我代表全校师生员工，向各位毕业生经过努力而顺利毕业表示最热烈的祝贺！

同时，借此机会，向那些辛勤工作无私奉献的老师们以及把心血和希望都寄托在同学们身上的家长们表示最诚挚的谢意！

今天是送别的日子，自古送别就有各种不同的情绪，有“晚风拂柳笛声残，夕阳山外山”的凄婉；有“劝君更饮一杯酒，西出阳关无故人”的苍凉；有“孤帆远影碧空尽，唯见长江天际流”的惆怅；更有“桃花潭水深千尺，不及汪伦送我情”的潇洒。

今晚，我们是兼而有之。在你们即将上路的时候，我希望你们多保重，不管你们将来身在何处，中国计量学院将永远和你们在一起！

明天，你们就要上路了，你们的前方是一个陌生的天地。校园里的路虽然狭窄，却自由通畅。校园外的路虽然宽敞，却人山人海。

作为学长，在你们临走之际，我要叮嘱几句。

明天将面临什么？简单地说，就是生活与事业。在你们以往的经历中接受过很多的励志教育，这可能会使你热血沸腾、壮心不已。

但我认为更重要的是如何生活。

第一，活得顺畅。每个人都生活在社会中，有社会就有体制，有体制就有主流意识和主流观念。

尽管我们每一个人都可以有自己的人生观、世界观和价值观，也可以有自己的处世哲学和处世态度，甚至可以在一定的空间里特立独行，但是，只有当我们尊重和适应社会的核心价值观，将自己融入体制社会的主流，才不至于被社会边缘化，才能活得顺畅。

第二，活得平和。作为社会的一员，每个人都要面对他人，你们要永远恪守善良、正直、诚信的原则。

当一个人遇上的是善待和仁慈时，才会懂得人间的温情，才会重情和善良。所谓的“仁者不忧”，一个人有了仁义的大情怀，其内心将无比仁厚、宽和，才可以不计较很多细节，不纠缠于小的得失，才能活得平和。

第三，活得通透。人生在世，当有慧眼，什么事情应该看透，什么事情无需看透非常重要。

一个什么都看不透的人，势必会碰到各种各样的障碍；而如果精明到骨子里、把任何事都看透的人，活得既累且没有情趣。其实，人的一生就是反复做一个动作：清零。一步一步走，一步一步扔，走出来的是路，扔掉的是负重，路越走越长，心越走越静，这样才能活得通透。

第四，活得放达。生活中总有失落和不如意的时候，最关键的是不管遭遇什么，都要有明朗的笑容，保持积极上进的心态，有面对苦难的意志和勇气。

人世间许多重要的事情，只有经过磨难和忍耐之后，才能够有所感悟，才能唤醒人性中的良知，才能活得放达，所谓“大其心则能体天下之物”。

第五，活得均衡。自然界给人们提供了风花雪月，等着你去临风踏雪、赏花弄月；人类文明的发展，给人们带来了艺术和体育，等着你去欣赏与享受。

在这个世界上，只要你愿意，就一定可以找到很多的兴奋点，它们可以让你的精神得到放松、心灵得到净化、生活更加充实、情趣变得高雅，会使你真正体验到更干净、更真实、更纯粹的生存。

作为毕业典礼，除了讲如何生活，肯定也要谈到事业。当然，前提是你想成就事业。

第一，成功是相对的。每个人都想获得成功，而成功是相对的。

比尔·盖茨是成功者，做成一件具体的小事也是成功。天底下只有一个比尔·盖茨，微软的事情就让他去干吧，我们干点别的。把握住每个机会，莫因事小而不为，做好每件事，欣赏每次小的成功，这样大成功也就呼之欲出了。有道是“深山毕竟存猛虎，大海终需纳细流”。

第二，目标是首要的。做每一件事情都要有目标，事情可大可小，目标可近可远，尽早确定了目标，就可以拓展做这件事的有效期并有针对性地去努力。

时光倒回到31年前的1978年，当时国家结束了十年动乱，迎来了科学的春天，召开了全国科学大会。著名报告文学家徐迟写了一系列歌颂科学家的报告文学，如《哥德巴赫猜想》、《地质之光》、《在生命的漩涡中》等等。那时有一个插队知青对其中的《在生命的漩涡中》特别感兴趣，这篇报告文学对我国著名科学家、北大校长周培源从事科学研究和人生的轨迹进行了描述，文章中自然界和社会的漩涡深深吸引了这位青年，并使他确定了将来从事漩涡研究的目标。

于是，当年他在报考大学志愿时，就填了与漩涡相关的力学专业；在后来分专业方向时，又选择了与漩涡更加相关的流体力学专业；当他读完了本科和硕士之后，更是考取了周培源先生的博士研究生，从事漩涡湍流的研究，实现了自己的意愿，达到了自己的目标。当初的那位插队知青就是现在已步入中年的我。

第三，学习是必要的。按时兴的话说，就是终身学习。

英语里也有一句话：“Stay Hungry, Stay Foolish”，即“求知若饥，虚心若愚”。仅凭过去所学的知识是无法成就事业的。

现在人类的知识总量，大约3年就翻一番，要有所作为，就必须不断地、踏实地学习和掌握新的知识，以此完善和充实自己。

第四，勤奋是基本的。有了目标，也知道了学习，但还必须勤奋。

不管一个人天资如何聪颖，要想达到一定的境界，离不开勤奋。在这里我不想论述勤奋与成功的关系，只想说发生在本周一美国NBA的事。

美国2008－2009赛季的NBA比赛结束了，洛杉矶湖人队获得了总冠军，科比是主要功臣。不管在座的是否是“科蜜”或者“科黑”，科比的勤奋是有目共睹的。科比是NBA现役球员当中得过常规赛MVP、全明星赛MVP、总决赛MVP以及4次总冠军的两个人之一。他能获得这些荣誉与他的勤奋密切相关，即便是现在已经达到了很高的水平，技术非常全面，篮球智商非常高，他的训练还是要比别人刻苦。所以，勤奋是基本的。

第五，挫折是难免的。在你将来的人生旅程中，你希望实现自身的价值并以此来回报社会和丰富自己。

然而，困难和挫折总是难免的，愿望与现实往往存在阶段性的差距，此时要能掌控自己，要沉得住气。正像张韶涵唱的，“每一次都在徘徊孤单中坚强，每一次就算很受伤也不闪泪光”。

第六，恒心是必要的。无论你是继续学习还是参加工作，在初始的一段时间里往往会不适应、不满意、有悔意。

这其中除了客观因素外，还有可能源于人性的弱点和认识的误区，比如“围城”效应。此时，要仔细分析，慎重决策，不要轻言放弃。

古希腊哲学家苏格拉底曾要求他的弟子每天做若干个甩手动作，一直坚持下去。许多年以后，坚持下来的只有一个人，这人就是与他同样有名的柏拉图。

一切成功都源于坚持，真正有成就的人善于把激动和兴奋转变成一种持久的努力。

同学们，说再见的时候到了，匆匆的你将走了，正如当初你匆匆的来，你挥一挥衣袖，不带走一片云彩，带走的却是量院师生的牵挂。当你再一次走出校门的时候，学校就变成了母校。

母校是什么？那就是曾经的家。希望你们能记住王安石的诗“春风又绿江南岸，明月何时照我还”，有空常回家看看!

谢谢！

2009年6月
