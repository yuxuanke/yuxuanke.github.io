CCTV3.15，反手就是一记大耳光

发表于 2017-03-19   |   分类于 生活百态   |     |   次阅读

引言：

3.15晚会每年都要出点纰漏，那年试图收拾麦当劳，结果是给人家做了活广告，再一年试图敲打苹果，结果让苹果变得更火……你想指责人家双重标准，没想到自个儿栽在了双重标准之下。

怎么说呢，麦当劳的内部品控标准，和苹果的售后服务政策，都要高于国家标准，更远远高于市场平均水准，不是说它们没有问题，然而放着那么多毒大米、毒蔬菜、地沟油不打，放着华强北的山寨货不抓，是不是有些舍本逐末？

与其说是被对手击败，不如说是自己作死。3.15的信誉真正一落千丈，还是栽在那著名的“8点20分发”，台上义正词严地打假，台下却找水军掺假，知道真相的吃瓜群众，从此粉转路人。

打脸

说到今年的败笔，无疑是无印良品。

“身经百战”的央视调查记者，在店里撕开了中文标签的一角，看见了“販売者”后面的地址，便如获至宝，直接就把无印良品钉上了“进口核污染地区食品”的刑柱。

没想到剧情一夜反转，无印良品的品牌影响虽然远不及麦当劳苹果，但这份硬气确实没给企业形象丢脸，直接贴出了严正声明，并附上原产地证明、出入境检疫等官方文件，干净利索地把耳光抽了回去。

当然这风险很大，你赢了央视一时，央视可以收拾你一辈子，还想不想在中国混了？

以往也有被冤打过的企业，也只敢谦虚低调地辩解，而罕有正面迎敌啪啪怼回去的，单凭这一点就值得点赞。躺枪者本来就不该认命，犯错者才该低头道歉。

“身经百战”的央视调查记者，犯了一个低级失误，日文中“販売者”的意思是销售商，与产地是两码事。

这并非日本特例，在我国任何产品标签中也司空见惯，要打假，也得有个起码的业务门槛，是不是先学会阅读商品标签？

另外3.15有个不好的习惯，欺软怕硬，只打表象，不打根源，只抓民企，不问官方，在本次涉嫌进口核污染区食品的曝光中，淋漓尽致地体现了这一点。

众所周知，我们拥有严格的海关，手腕子上多戴了几块表，都能罚得你哭不出来。自打日本核泄漏事件之后，我国两次升级禁运名单，并要求所有日本国进口食品出具原产地证明，限制标准比其他国家都严，对一个遍地有毒食品的国家来说，倒是有点滑稽。当然核心问题在于，市面上出现了疑似污染禁运区的食品，难道首先打的不是海关、出入境检疫的脸？央视调查中为何缺席这一章重头内容？

当然这是惯例。3.15曾经打过数不清的问题食品，但是从来没有问责过食药监局，曾经怒对过很多外企、名企，却从没有追究过工商监管，就连本次“三无体检队”公然窜入几百所小学行骗，当地教育局的职责何在，也问都不问。

打铁先得自身硬，在民企、外企面前是好汉，在有关部门面前是软蛋，一个逮住往死里打，一个是打死了也不敢问，这点风骨连大裤衩都不配穿。

也就只有企业害怕央视、害怕3.15吧。

昨日上海出入境检验检疫局也干脆利落地回应，证明无印良品没有来自于辐射地区的产品，还把央视不会读标签的低级失误，又明确重申了一遍。虽然央视没有点检疫局的名，但检疫局也绝不背这隐隐约约的锅，这不是大水冲了龙王庙，而是强虎碰上了霸王龙。

一台以打假为主题的盛会，如果要保证客观中立，似乎应当是一台公益栏目，然而晚会之前冗长的广告，证明这是一台商业气氛浓郁的晚会，而且迄今为止，还没有出现把会前上广告的企业给打了的先例，这样的“低级失误”倒是从来没有犯过，于是会前广告被视为“安全区”……那做贼心虚的无良企业还不排着队送钱去。

打假确乎有利可图，然而官方却严禁民间打假人入行，成为官方媒体的独家生意。当消费者没法自己维护自己的利益，其垄断代言人却可借此渔利，媒体行业充斥有偿新闻、有偿不闻，越往上越登峰造极。

屡遭诟病的3.15，能否认一回错？
