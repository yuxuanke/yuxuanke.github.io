2月份的curt，都在干嘛

发表于 2017-03-01   |   分类于 生活百态   |     |   次阅读

2017年3月1日，今天可算把自己静静的摁在了浙大图书馆里，看着YY920荒废的一个多月，对自己更多的愧疚与不安，每周末，内心都很煎熬。

这一个月，可谓是一年中最漫长忙碌的一个月。

第一次春节去了她家以及带她来我家、杭州最好的朋友“夜江”选择回家、大学最好的兄弟“明叔”结婚了、高中三年基友“大胖”也结婚了、我的那个她研究生毕业了、新房开始装修了·······

2017年，更多的是鞭策。

YY920.cn，是curt对生活的一种态度，3月，重拾那颗不求安分的心。
