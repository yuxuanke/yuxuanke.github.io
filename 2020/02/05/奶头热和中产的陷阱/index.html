奶头乐和中产的陷阱

发表于 2020-02-05   |   分类于 中国社会   |     |   次阅读

1978年中国开始改革开放，10年为一个周期，经历纷杂激荡的40年，而2008-2018，甚至现在2020年，我们经历着80、90后最疯狂的10年。

此时此刻，我们正经历着03年SARS-CoV之后，再一次传染性更强的2019-nCoV新冠肺炎，肆虐全国。

奶头乐理论

由于生产力的不断上升，世界上的一大部分人口将不必也无法积极参与产品和服务的生产。为了安慰这些“被遗弃”的人，避免阶级冲突，方法之一就是制造“奶头”。

喂之以“奶头”——使令人陶醉的消遺娱乐和充满感官刺激的产品（比如网络、电视和游戏）填满人们的生活、转移其注意力和不满情绪，令其沉浸在“快乐”中不知不觉丧失思考能力、无心挑战现有的统治阶级。

“奶头”的形式有两种：

  • 一种是发泄性娱乐，比如开放色情行业、鼓励暴力网络游戏、鼓动口水战。

  • 一种是满足性娱乐，比如肥皂剧和偶像剧、小视频、麻将棋牌、播放真人秀节目、手游等。

中产阶级

“中产人士”的收入比上不足，比下有余。

自我感觉挺良好，由于有一点闲钱和闲时间，中产们开始用一种比较廉价的方法模仿起了富裕阶层的生活方式。

例如，消费奢侈品牌、出国度假、各种讲求生活格调以及追求所谓的精神生活。

把大部分宝贵的业余时间浪费在“美好的事物”上面，却忽视了继续提高自我追求、更新自我知识，以便适应不断变化的新经济环境。

在社会渗透性的主流消费价值观引导下，消费升级，超前消费，过度消费。

即使在收入没有明显增加情况下，依然在盲目追求消费升级，增加消费支出。

甘愿成为欲望和金钱的奴隶。

中产阶级，具有的三大幻觉

  • 1、认为自己有一定的资产（不动产为主）就可以高枕无忧。

  • 2、认为自己有一定的社会地位，说话有影响力。

  • 3、认为勤奋就会成功。（参照二八定律，非线性思维）
