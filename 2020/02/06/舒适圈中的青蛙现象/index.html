舒适圈中的青蛙现象

发表于 2020-02-06   |   分类于 工作与职场   |     |   次阅读

一些突变事件，往往容易引起人们的警觉。而导致人于死地的却是在自我感觉良好的情况下，对实际情况的逐渐恶化，没有清醒的察觉。

在2003年网购这个新鲜消费方式出现的时候，那个当下的我们都没有想到，十几年间会带来物流、仓储、线上商城的异军突起，对线下实体来了一次革命。

在2008年触屏手机问市的时候，那个当下的我们，是否有想到我们正在经历移动和智能时代的革命，不，那个时候我只沉迷在水果忍者游戏中。

在2017年抖音开始进入大众视线的时候，那个当下的我们，可曾预想到我们正在进入移动端的小视频和直播时代，不，那个时候我被逗趣小视频带来的短暂快感所麻痹。

罗振宇在2019年《时间的朋友》演讲中，提到宏观是我们必须接受的，微观才是我们可以有所作为的。

大部分人都可以感知到宏观的变化，比如对近3年、甚至5年的社会发展感知；

但对于微观的变化却熟视无睹，比如今天的一次聚会聊天、昨天的一项新技术投产应用新闻、前天的一个政策推出。

正是每天发生的一点小变化，汇集成我们能够感知的小趋势，最后成为社会的大趋势、大变革。

说到青蛙现象，大家都耳熟能详，其实自己就是那只青蛙。

大环境的改变能决定你的成功与失败，然而当下大环境的改变又是看不到的，所以我们必须时时注意学习、警醒、欢迎改变，才不至于太迟。

常说的中年危机，何尝不是因为待在自己的舒适圈里太久，温水煮青蛙，等到危机来临时，才发现已经来不及了。

人类天生有一种惰性，不到迫不得已就不会去改变现行的各种还过得去的做法，当这种做法还能够让人得到很大的满足时尤其如此。

太舒适的环境就是最危险的时刻，也许你已习惯的生活方式，就是你最危险的生活方式。

在平时，要能觉察到趋势的小改变，就必须停下来，从不同的角度来思考，学习是最能发现改变的最佳途径。

青蛙现象/法则：

他们将一只青蛙放在煮沸的大锅里，青蛙触电般地立即窜了出去，并安然落地。后来，人们又把它放在一个装满凉水的大锅里，任其自由游动，再用小火慢慢加热，青蛙虽然可以感觉到外界温度的变化，却因惰性而没有立即往外跳，等后来感到热度难忍时已经来不及了。

生于忧患·死于安乐

未雨绸缪·危机意识
