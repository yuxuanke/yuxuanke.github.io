《前任攻略3:再见前任》观后感

发表于 2018-01-06   |   分类于 生活百态   |     |   次阅读

再见，前任，不是让我们去怀念以前，而是珍惜当下，那个愿意陪你一起的现任。

最先想去看这部电影的念头，来源于抖音。有一次在西溪银泰吃完饭，我本是一句玩笑话，我们去看前任吧，电影院。

以为她会拧我，因为我是她的初恋，而我却有好几个，本是一句找打的挑逗，结果她却来了一句，可以呀。

我当时还是拉着她回家了。

有一个晚上，她说，我买好票了，5：50分，晚上一起去看电影。

分享下电影台词

现场，她哭的像个泪人，我却毫无泪点；我明白，我只有失去她，才会像孟云在广场上哭的一塌糊涂，七年多的陪伴，仅此一份。

林佳:“你不要我了怎么办”

孟云:“那我就像至尊宝一样去最繁华的街道喊一百遍 林佳我爱你”

孟云:“那你不要我了怎么办”

林佳:“那我就吃芒果 吃到死为止”

假如真的有那一天，那我就像至尊宝一样去计量第一次表白的启明广场喊一百遍，我爱你

别等失去才知道

陪伴和安全感对女生有多重要。

“我一直给林佳描绘未来，却不知道她在意的是现在。她需要从现在来确定未来。”

借着吵架的梗，撒着思念的娇

谁没有丢掉过一个人，只因为一直幼稚的抓紧那点自尊；谁没有玩命的爱过一个人，最后再也没有力气那样爱别人。

七年之痒，在这个2017年的9月后，不管我承不承认，的确在我们之间挠痒。

你有了你的忙碌工作和坚持，我有了我的理想生活和取舍。

“一个以为不会走，一个以为会挽留。缘分自然就到了头”

别相信什么走散了也叫命定，感情从来事在人为。要知道最无用的就是倔强，可很多人却最喜欢拿来用。

我再也不会用我有多么辛苦，我是这个家的顶梁柱，我要赚钱养家这些看似冠冕堂皇的理由，来为自己找借口了。

总在错的时间遇到对的人总在失去后才明白应该如何去爱一个人你陪我成长，然而却不是你陪我到最后!

我们都是陪伴彼此一起成长的，见证了彼此最好的芳华，希望也是陪TA到最后！

我走到那个路口前，只要你说我爱你，我就马上跑回你身边。

这是一个彼此尝试放手的好方式，好聚好散，就算放手，也要像开始时的浪漫。

因为我从她的眼睛了看到了那种无谓，像当初的我一样。

我不想有这么一个人的出现，有些人和事是无法替代的。

记住，在男人看来，自己的女人不接受任何人的献媚，才是一个女人表达爱最感动的方式。

看到这句话，我莫名的认可。

关于电影伊始的吵架

在知乎上看到这段话，我感觉笔者是个过来人。

不知对错的想法

在以后的日子里，不管工作多么忙碌，生活多么富足或潦困，记住那个陪伴你的人
认真的回答彼此的每一句话、用心的记下彼此的每一个想法、理解的尊重彼此的每一个决定、宽容的接受彼此的每一个缺点。
不要尝试着去改变TA，你可以影响着让TA快长大和更成熟；不要让TA变成你理想中的那个TA，不然，你看到的都是缺点，慢慢发现你不再爱了。
