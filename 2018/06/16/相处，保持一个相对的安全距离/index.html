相处，保持一个相对的安全距离

发表于 2018-06-16   |   分类于 生活百态   |     |   次阅读

每个人，都有一个潘多拉的魔盒

潘多拉魔盒（Pandora’s box），魔盒，释放出人世间的所有邪恶——贪婪、虚无、诽谤、嫉妒、痛苦等等，当她再盖上盒子时，只剩下希望在里面。

每个人，都有这么一个魔盒，只不过在正常状态下，很好的封存在内心深处，不曾触及。

隐私，是一个人不愿意向他人公开的隐秘过往和内心经历，而这些，就藏在这个魔盒里。

一段阴暗创伤的童年过往、一个卑微不堪的家庭环境、一段沉沦沮丧的生活时期、一场挚爱深刻的感情失利、一个唾弃羞耻的自私抉择，都可能让你筑起高墙，不愿提及或承认。

这些，应该被尊重

所谓的隐私权，只要这种经历不包含损害他人的情节，任何与此经历无关的人，都无权过问，更无权强行公开。

有时候，一个人会有向人倾诉内心的愿望，但这种愿望的发生往往取决于特殊的情境和心境，强求不来。

在情侣之间，最可怕也最可笑的侵犯莫过于以爱情的名义，强求对方向自己敞开心灵中的一切。

相爱的两个人，应该保持一个相对的安全距离，最重要的是两颗灵魂本身的丰富以及由此产生的互相吸引，而绝非彼此的熟稔乃至明察秋毫，始终保持一个尊重和包容的态度。

这个相对距离，体现的是尊重，是包容。

你要学会，尊重对方的人格、尊重对方的家庭，包容对方心里留存着一些难以启齿的隐私，心照不宣。

所以说，在爱情中，并不全是毫无保留，言无不尽。在信任的基础上，有所保留，独自承受，让双方留有一个安全距离，不乏是高情商的体现。

不是毫无保留，而是有所保留

曾经，我犯了一个严重的错误。

以为毫无保留的沟通，转达和表达真实想法，就是尊重。错了，在没有考虑对方感受的言语，尽是自私。

距离产生美，不要打翻了彼此的潘多拉魔盒，爱一个人，不是毫无保留，而是有所保留。

成人的世界里，没有对错，只有立场。

最近，听说了一个名词“原生家庭”。

原生家庭指的是你从童年开始成长的家庭，是相对于成年后组成的婚姻家庭而言的——即你和你的父母、兄弟姐妹或祖父祖母、继父继母等成员共同生活的家庭。

我们的很多技能、习惯、处事方式都是在原生家庭中习得的，包括如何与人沟通、互动，处理我们的情绪和需求。

同样，我们价值观和信仰的形成也与我们的父母息息相关。原生家庭中的亲属及其他关系对我们有着至关重要的影响——你在什么样的家庭出生、成长，直接决定着你成年后会成为什么样的人。

再者就是另一个名词，“原生家庭问题”，这个普遍存在于每个家庭，取决于这个家庭人员组成的综合素质，父母也是凡人，人无完人，并非无所不能。

所以说，每个人辨识出来自原生家庭的问题是相当重要的。

清晰认识到原生家庭带给自己在习惯、为人处世、甚至价值观上的不足和缺陷，并加以纠正，是我们组建自己家庭关键的因素。
