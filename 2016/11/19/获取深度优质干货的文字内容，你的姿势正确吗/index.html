获取深度优质干货的文字内容，你的姿势正确吗?

发表于 2016-11-19   |   分类于 阅读   |     |   次阅读

阅读这篇文章之前，curt想先问你一个问题：

当你有2-3个小时时间，想阅读一些有质量有深度的文字内容时，你首先想到的是什么？（什么途径/方式）

请你思考10秒钟，记下你的第一反应。

  1. 深度的、优质的、干货的文字内容
  2. 朋友圈的采访
  3. 书籍获取
  4. 移动端获取
  5. 电脑端获取
  6. 培养专注高效的获取习惯

1、深度的、优质的、干货的文字内容

在现在这个爆炸性的信息社会，新闻是免费的、视频是免费的、音乐是免费的、图书馆书店的书是免费阅读的、手机各类App阅读是免费的、各种信息大都是免费的。

但是在这么多海量的信息里面，发掘出真正对『你』有价值的信息，本身就是很有价值的事。

所以说：

免费的，也许是时间成本最高的。

信息的免费，则需要你花费更多的时间去筛选。服务的免费，则需要你花费更多的时间去等待。

暂且，curt把对『你』有价值的信息，定义为深度的、优质的、干货的文字内容。

80、90后，正处于奋斗打拼的年纪，大部分是没有时间和精力静下来记录些带有思考性的文字内容，那他们平时是如何获取深度优质干货的文字内容的呢，这显得尤为重要！

2、朋友圈的采访

抑制着忐忑好奇的心情，curt以微信单聊的方式，直接采访身边29位好友（开篇提问），以他们第一反应的回答为主，凡是提到的途径/方式均作为一票。

统计结果如下图：（29位好友，年龄分布在83-93区间）

虽然样本基数很少，但还是能折射出不少可分析的亮点，请问你的答案，中招了吗？

3、书籍获取

读书无嗜好，就能尽其多。不先泛览群书，则会无所适从或失之偏好，广然后深，博然后专。　——鲁迅

读书破万卷，下笔如有神。——杜甫

将近50%的比例，认为看书是获取有价值文字内容，最为有效的方式。

书，是人类记录和传承知识最为原始的方式，一本书有着系统全面、简单安静的特点。

一本书、一对桌椅、一杯咖啡，就可让你暂且忘记生活的周遭，静下心来阅读。不像电子设备，有着电池续航、信息电话、网络诱惑、碎片式信息等因素的打扰。

虽然现在书籍品质充斥着良莠不齐现象，但相比网络海量的信息，书籍依然是你筛选有价值内容最为高效的方式，你可以快速筛选你感兴趣方面的书籍，浏览目录和引言判断其是否优质，并作为“枕边书”随时阅读。

这也说明了，像逻辑思维这样以推销畅销书的方式，为什么会获得成功，因为他替我们筛选了高品质的书籍并推荐给广大读者。

看书，是你主动筛选信息，进行短期系统化学习的过程。

4、移动端获取

第二梯队，是通过移动端各类App获取有价值的文字内容，其中微信公众号和知乎最为常见。

微信公众号

为什么近3年微信公众号会这么火？

curt的解释是：公众号帮你做了一件事，就是替你筛选出最优质的内容和最关心的信息，提高了你的效率，解决了你的痛点。

门户网站、搜索引擎、微博博客，这些都是5年前特别特别流行的东西，但是现在正一步步陨落，被80、90后边缘化。

为什么？

门户网站的特点是大而全、搜索引擎的特点是快速筛选、微博的特点是高效新鲜，这摆在5年前都是优点，但是在当下，这可能痛点。

当下的80、90后，由于生活和压力和节奏，更需要的定制化和差异化的服务。而门户网站铺天盖地的信息、搜索引擎筛选出广告劣质内容、微博快餐式信息消费，都已经让年轻人厌倦，因为要从中找到对自己有价值的文字内容实在是低效，1个小时的时间消逝即去。

以上就是微信公众号成功的原因，各类提供定制化专题差异化内容的公众号，如雨后春笋般。

知乎

知乎，作为国内最大优质化内容的聚集社区，由于吸引了众多专业人士、意见大牛的加入，使其问答内容变得更为优质。

curt本人也是混迹于知乎，时常搜索相关的有价值信息，的确，相比于搜索引擎，你会获取到意想不到的内容，有深度有见解的回答内容比比皆是。（关键没有广告）

知乎，curt认为是目前较为成功的问答搜索引擎，知乎因其纯粹的内容社区和严密的内容管理，更加简约且优质，相比传统的百度搜索引擎，不会出现一个关键词搜索，前面3-5个链接都是广告推广，后面大部分是网页新闻等劣质内容的现象。

（在知乎出来之前，豆瓣是做的最好的，现在豆瓣主要是影评和书评做的比较好。所以在你想了解一部电影、一本书的品质时，建议上豆瓣较为高效。）

其他App应用

作为移动端，定位阅读的App也分两类：

  • 小说阅读、阅读器：将电子版的书籍下载至手机端，进行阅读。（Kindle相当于一个没有网络的纯粹阅读器，远离了移动端的众多诱惑。）

  • 为用户提供优质短篇文字内容的阅读器：如一览、简书、知乎等电脑端和移动端同时经营的平台，首先为用户筛选过滤优质内容，用户可根据需求进行定制化的关注订阅，提供差异化服务。

移动端的文字内容获取，一定程度上是你，被动的接受碎片化信息的短暂学习过程。

移动端因其便捷性，可以随时随地的获取，很大程度的利用你碎片化的时间，但正是这些特点，在你获取的过程中，极其容易被打扰（一条微信、一句话、一个举动）。

碎片化时间获取的也是碎片化的信息，你很难系统的学习并吸收，更像是快餐式的信息消费，看山是山，看水是水。

对人生境界的阶段描述中，有一种很装逼的说法：

看山是山，看水是水；

看山不是山，看水不是水；

看山还是山，看水还是水。

第一重因为无知，第二重因为信息轰炸，第三重回归初心。

5、电脑端获取

少部分人，认为电脑端的网页、论坛、网站是获取有价值文字内容的有效途径。

虽然在电脑端使用时，面临的诱惑也很多，但因其天然的操作便捷性优势，只要通过合理高效的方式，定能收获事半功倍的效果。

  • 搜索引擎而言，谷歌搜索绝对是首选，原因有两点：一是精准的内容搜索，二是没有防火墙的和谐。
    （谷歌搜索服务已退出中国，如需使用需翻墙操作）

  • 优质论坛、博客、平台等小众网站，在互联网的海洋里，其实隐藏着不少优质的小众网站，里面聚集着一群有情怀、有深度的内容生产者。
    （这些网站，不做列举，毕竟没有广告费）

  • 电脑端网页浏览，有着快速高效的筛选和阅读优势，可随时借助备忘录和EverNote等软件保存整理素材，并快速查询延伸信息，形成较为全面化的学习。
    （长期形成自己的素材库，方便随时调用）

电脑端的文字内容获取，是定向进行积累与全面学习的过程。

6、培养专注高效的获取习惯

一年365天，一天24小时，请问你，有多少时间是留给自己获取有价值的文字内容，自我充电？

5个人，同样玩着手机一个小时，A在和朋友聊天，B在玩手游，C在浏览新闻八卦，D在阅读优质的文章，E在掰玉米，每样都有涉及。

当你想获取对『你』有价值的信息时，请给自己培养一个获取方式的好习惯，做到专注高效。

当你有2-3个小时时间，想阅读一些有质量有深度的文字内容时，你要尽可能的将这2-3小时都用在学习吸收上，而不是浪费在准备、筛选上，让效率最大化。

当你选择微博、朋友圈方式获取文字信息，殊不知，这样的方式是最低能的，你在做的无非是消遣时光。

生活中可悲的一小撮人，将打发消遣时间的方式，误以为是在自我充电（学习）。
