几个玩音乐的硕士博士生们|鹿先森乐队

发表于 2016-11-08   |   分类于 生活百态   |     |   次阅读

一首《春风十里》，听着就想单曲循环，脑补着属于自己的故事，却撩遍了全国的文艺青年。

在朋友圈曹妹子分享下，curt第一次接触到这首歌，向来对音乐五音不全又不太感冒的curt，却被这首歌的旋律瞬间俘获，单曲循环。

细品歌词，记忆的画面怦然翻动，curt自认为也不是一个饱有故事的工科男，却已不能自拔。

我在二环路的里边想着你

你在远方的山上春风十里

今天的风吹向你下了雨

我说所有的酒都不如你

我在鼓楼的夜色中

为你唱花香自来

在别处沉默相遇和期待

凭着对这首歌的着迷，对这个鹿先森乐队也产生了浓厚的兴趣。乐队在2015年8月31日成立，由5位团队成员组成，10月份发行第一首单曲《春风十里》，开始被大众所了解。

看着TA们的介绍，各自从事着与音乐不着边际的正业，却因为音乐的共同爱好，正朝着梦想一步一步迈进，就如太阳，光芒不可阻挡。

在2016年，新加入一位PP鼓手，由此圆满完成乐队的组建。

未来生活，是因为梦想，而不是将就。

工作之后，curt发现身边有人白天做着重复厌倦的工作，夜晚却有一个声音在呼唤TA改变。

在鹿先森乐队，看到了一群不再年轻的少年，因为曾经的音乐爱（梦）好（想），聚首，实践，不为功与名，只为听从内心深处的那个声音。

幸运的是，倍倍（鹿先森乐队队长）身边能聚拢一群志同道合的小伙伴，curt对此羡慕不已。

当下的大势洪流，每个人都无法独善其事去选择自我，一开始就是要为面包和牛奶努力奋斗，也许是10年、20年、30年，或者是一辈子。

鹿先森，选择把工作作为支撑生活的方式，将乐队作为实现音乐梦想的方式。

这样，至少不会在追逐梦想的路上而穷困潦倒，假使有一天，梦想实现了呢？

curt的梦想是有一家不以此为生的咖啡馆，在里面，安静的向别人述说我与她的故事。

不要小瞧每个人的小梦想，那也许就是支撑TA为之奋斗的动力，也许目前的努力与梦想毫不相关，但总有一天，看到，触摸到。

那个努力和梦想早已约定好的交点。
