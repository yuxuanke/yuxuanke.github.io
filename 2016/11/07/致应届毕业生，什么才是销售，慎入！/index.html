致应届毕业生，什么才是销售，慎入！

发表于 2016-11-07   |   分类于 生活百态   |     |   次阅读

引言

不避讳的承认，curt也是一名普通的销售（工业型）。

一提到销售，相信大家脑海里都会浮现一个典型的销售员形象，如果让你用几个形容词来形容，最可能出现的是烦人的、能说会道、不诚实、脸皮厚、笑脸相迎、西装革履，更进一步还要额外加上“关系钻营”“尔虞我诈”“幕后交易”这样的词汇。

人们对销售已经形成了固定的认识，“销售”的污名化，给销售行业的从业人员造成了不少心理障碍。

那什么是销售？

  • 作为一种职业，TA是介于生产者和消费者之间的纽带。
  • 作为一个商业流程，TA是提供产品或服务，获取收入回报的过程。
  • 作为一种思维模式，TA是主动创造价值的方式。

也许绝大部分只能理解到第一层面，真正做过销售的才能领悟到销售思维的重要性，这个下次单独细说。

10-11月这个时候，刚好是应届毕业生找工作的季节。每个人都会面临选择什么样的行业和什么类型的职位，刚好有个研究生朋友和curt聊了他想去GE医疗器械销售岗位面试的想法，思虑再三，curt觉得有必要和大家分享下（工业口）销售前三年的工作状态。

前三年的销售工作状态

背景：假设一个0销售经验的人（应届毕业生）获得一个工业销售岗位，那么依据目前的社会现状，前3年，你需要做什么和能得到什么呢？

第一年

1.入职第1个月

基本上销售新人的第一个月都是学习和被培训中度过的，这个时期销售新人会被打鸡血，对市场充满憧憬，认为项目、客户，鲜花，美酒在等待自己去掠夺，迫不及待的想去市场去历练。

2.入职2-3月

这个时期已经去市场去历练和修行自己了，发现自己被培训的套路运用到客户身上好像并不起效果，但好在哪里，坏在哪里自己也不知道，反正每天就是按照公司的要求机械的去拜访，至于见到客户说什么和做什么，按公司的培训去做就好，反正自己是一片空白。

3.入职第4个月之后

入职第4个月之后，忽然发现自己是穷逼，每月的工资捉襟见肘，看其他的销售同事开销好像都很有钱的样子，为什么自己的工资自己每个月都不够花？一般而言，从第3个月起，上级领导开始对自己谈销售任务，从第4个月开始要求销售任务，并给自己压力。

正常的话，在第5或者第6个月，销售新人有了第一个合同，这样就又被打鸡血，当然第一个合同如何签订的，自己可能都迷迷糊糊，脑海里是绝对没有控单和操作的概念，公司怎么要求就怎么做，于是就签订了订单，销售新人会萌生“销售不过如此”的想法，但是自己虽然充满憧憬，但是过年的时候却忽然发现，自己的销售结局并不如自己想象那样，年底赚个5万10万，第一年的年底可能就领个5000元奖金回家了。

总之，做一个销售新人，第一年要做是的完全服从遵从公司即可。`

第一年能得到是什么？

一般而言，销售新人的第一年是几乎拿不到上万以上的现金回家过年的，第一年签订的合同总数也就是1-2左右，老板也一定告诉他，货款没全部回收，所以提成不能全发，只发个5000元回家过年。

自己的总收入一般是：自己工资 + 5K

第二年

1.第3-6个月，继续开发客户，但这个时候开始有了抱怨

销售新人过了春节又到公司去报道了，公司一阵春节例会宣传动员，新人又开拔前线，由于第一年没赚到钱，家里人可能会说，其他人可能会议论，都让销售新人感觉不足，人一旦感觉不足，就会抱怨，于是销售新人开始了职业上的抱怨和对赚钱的强烈的企图心。

2.第6-12月试图从1个客户那里混点小钱

人一旦有了赚钱的企图心就会左思右想跃跃欲试，但是公司制度和混钱的细节自己并不掌握，所以会患得患失有贼心单没贼胆，一般而言，机智的销售新人会从自己的客户里找个关系到位混钱或飞单，忠厚的销售新人会有想法但是不敢去实施。

总之，作为一个销售新人在他的销售第2年，会签订3-4个订单，会给公司赚到钱，自己能不能赚到钱要看自己的胆略和客户的掌控程度。

第二年能得到什么？

1：第二年由于掌握了开发客户技巧，所以会得到2-5个关系不错的老客户。

2：过年的时候，老板一般会对他诉说公司的困难，请求他的理解和支持，过年的时候一般会发2-3W元左右的提成。

自己的总收入一般是：自己的工资 + 2-3W提成

第三年

1.第3-6月继续开发客户，但冷眼看公司的政策

销售新人经过2年的折腾，发现自己的一腔热血却换不来自己想要的金钱，于是热心慢慢成为冷血，对公司政策的更迭和人员的来去已经毫不在意，自己已经彻底的盘算自己，关注自己。

2.第7-12月，一定会飞单，卖单，贴牌自己赚点小钱。

做了2年，听了很多，看了很多，也学习了很多，第3个年头，终于明白这个世界只有自己才能救自己！于是继续老实或表面装老实，但是背地一定会亲自飞单卖单贴牌或兼职，开始了一系列的自己捞钱的行为，从此开启了销售人员的利用公司的资源捞钱之路。

销售新人这个时期已经不装高尚了。

总之，销售新人在第3个工作年头，已经满身风尘，看淡红尘，对公司失望多于期望，开始了捞钱之旅，从此一生有机会能捞就捞，开始创富。

第三年能得到什么？

1.一个公司能生存3年的销售，已经算是老人了，所以在公司内部会得到不错的人脉。

2.过年的时候，老板会多给1w上下的红包，但仍然会告诉他公司的困难，和对他个人的认可，会许诺高管位置，但就是不提提成兑现。

自己的总收入：工资 + 自己操作得来的见不到光的10W 以上

三年后，你的蜕变

一般而言：一个销售员从0开始做销售，第3年为分水岭，如果第3年能有10万的纯收入说明自己适合销售，如果纯收入低于10万，说明自己不适合销售，或者说心态和技巧出了问题，急需提高。

有的销售菜鸟会问，“有些事情违法啊！”。

ok，当你路遇1个红灯，这个红灯方圆几十KM，除了你都无任何人和车，但是你受的教育是红灯停，于是你遵守红灯，哪怕这个红灯坏了，你都会等的无怨无悔，你的心情是光荣的，感觉自己是受表扬的，自己觉得站在道德高地并洋洋得意的，你的行为是对的但不是“最有效的”。

销售是一门追求“最有效”而不是“道德最正确”的职业。

红灯的比喻是不恰当的，但是人生的长跑中，一步慢，步步慢。所以，犯罪的事情坚决不做，违法的事情视代价大小为之，譬如你可以赚急需的1000万，但是付出的代价上法院的“老赖黑名单”，从此你不能乘坐飞机和夜总会潇洒，你会如何取舍？

有道德洁癖的人是不能做销售的，因为他没冒风险的胆略和气魄，前首富黄光裕的发家是沿铁路火车贩卖电子表，而“长途贩卖”在当时是违法的，是“投机倒把罪”，现在社会贩卖物品是正常的不能再正常的生意，而在过去则是违法犯罪。

所以，一个循规蹈矩万万不敢逾规的人是不适合做销售的，销售某种意义上说，是最大化自己的边界，而社会和道德又紧密的编织一个边界，把我们困在界内，销售就是在社会和道德编织的囚笼里挣扎，把自己的边界最大化，所以有时会触碰社会和道德的边界的。

销售中回扣是真实存在的但又违法的行为，你敢不敢送回扣？只要一送，你就违法了！犯罪了！

有多少风险，就有多少利益，这句话大体是对的！

对于工业行业的销售而言，curt总结为：

销售，是一个需要时间积累和沉淀的职业；销售，是一份考验你已有道德底线的岗位；销售，是一种深入侯门深似海的工作。

这就是工业口的销售，如果你是一个应届毕业生，正在考虑投身于销售行业，看完能有所领悟的，说明你可以尝试，如果不知所云，那请远离销售吧。
